# Project: Campino Agency

Campino Agency is a Persian RTL Laravel website for a 360-degree advertising campaign agency.

## Goal
Build a professional lead-generation website for an advertising campaign agency. The website should explain services, advertising funnels, platforms, pricing, case studies, and collect campaign requests.

## Tech Stack
- Laravel
- Blade
- Tailwind CSS
- MySQL
- Filament Admin Panel later
- Persian RTL UI

## MVP Pages
- Home
- Services
- Funnel Design
- Advertising Platforms
- Case Studies
- Pricing
- Blog
- Campaign Request
- Contact

## Writing Tone
- Persian-first
- Simple and sales-oriented
- Natural Persian
- Avoid overly formal words like "باشد" when possible
- Use clear CTA-driven copy

## Coding Rules
- Use clean Laravel conventions.
- Use Blade components.
- Keep UI RTL and responsive.
- Validate all forms.
- Keep MVP simple.
- Do not over-engineer.
- Create reusable components where useful.
